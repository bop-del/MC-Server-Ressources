# MC Server Resources

A collection of resources for the Minecraft server, including resource packs and other assets.

## Contents

### `resource_packs/`
- **ExecutableItems_Pack.zip** — Resource pack for the ExecutableItems plugin, providing custom textures and visuals for custom items on the server.

## Usage

Download the resource pack from the `resource_packs/` folder and install it in your Minecraft client, or configure your server to send it automatically to players on join.
